# Pixel Tab Art

A Chrome extension (Manifest V3) that overlays a small, deterministic pixel-art sprite on every tab. Each tab gets a unique sprite derived from its URL, so you can identify tabs at a glance.

## Features

- **Deterministic art** — same URL always produces the same sprite (seeded from hostname)
- **Draggable overlay** — grab the sprite and reposition it anywhere on the page
- **Subtle pulse animation** — gentle opacity cycle while the tab is active
- **Popup controls** — regenerate art with a new random seed, or toggle the overlay on/off
- **Non-intrusive** — fixed 64×64 px, semi-transparent, no impact on page layout

---

## Loading in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the `pixel-tab-art/` folder

The extension icon will appear in the toolbar. Open any webpage and the sprite appears in the bottom-right corner.

## How the Pixel Art Works

### Seeded PRNG — mulberry32

All randomness is driven by **mulberry32**, a fast 32-bit seeded PRNG:

```js
function mulberry32(seed) {
  return () => {
    seed = (seed + 0x6d2b79f5) | 0;
    let t = Math.imul(seed ^ seed >>> 15, 1 | seed);
    t = (t + Math.imul(t ^ t >>> 7, 61 | t)) ^ t;
    return ((t ^ t >>> 14) >>> 0) / 4294967296;
  };
}
```

The seed comes from **FNV-1a hashing** the page's `hostname`. This guarantees the same site always gets the same sprite across browser restarts, with no storage required.

### Mirrored 8×8 grid

1. Generate the **left 4 columns** of an 8×8 logical grid. Each cell is assigned a palette index (transparent, primary, accent, or shadow) based on PRNG thresholds.
2. **Mirror** the left half to the right: `row = [...left, ...left.reverse()]`. This creates bilateral symmetry, making the sprite look like a recognizable character.
3. Each logical pixel is drawn as an **8×8 block** on a 64×64 canvas → crisp, no anti-aliasing (`image-rendering: pixelated`).

### Palette

Two hues are derived from the seed (offset ~50–120° apart on the colour wheel). Three opaque colours are used: primary, accent (lighter), and shadow (darker). Transparent cells create the silhouette.

### Regeneration

Clicking **Regenerate** in the popup generates a new seed via `Date.now() ^ Math.random()`, saves it to `chrome.storage.local` keyed by the page's default seed, and pushes it to the content script via `chrome.tabs.sendMessage`. The override persists across page loads.

node scripts/create-icons.js
