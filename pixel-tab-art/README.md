# Pixel Tab Art

A Chrome extension (Manifest V3) that injects a pixel-art critter into every tab. Each site gets a unique cat or dog that walks along the top of the page, so you can identify tabs at a glance.

## Features

- **Deterministic per hostname** — same site always gets the same critter type and fur color
- **Walking animation** — two-frame walk cycle along the top of the page, flips at screen edges
- **Hover to spook** — mouse over the critter and it turns around
- **Popup controls** — preview your tab's critter and toggle it on/off
- **Non-intrusive** — `pointer-events: none`, never blocks clicks or page layout

---

## Loading in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the `pixel-tab-art/` folder

The extension icon appears in the toolbar. Open any `http`/`https` page and the critter starts walking. Refresh tabs that were open before loading the extension.

## How It Works

### Identity

The page's `hostname` is hashed with **FNV-1a** into a 32-bit seed. The seed determines:
- **Critter type** — even seed → cat, odd seed → dog
- **Fur color** — hue, saturation, and lightness derived from the seed via mulberry32

The same site always produces the same critter, with no storage needed.

### Sprites

Each critter is a hand-crafted 8×8 pixel grid with two walking frames. Characters map to colors:

| Char | Meaning |
|------|---------|
| `.`  | transparent |
| `b`  | body (main fur) |
| `l`  | light (belly) |
| `e`  | eye |
| `n`  | nose |

Left-facing frames are generated by horizontally flipping the right-facing sprite data.

### Walk loop

Each `requestAnimationFrame` tick advances the critter by 0.65 px in the current direction. Every 10 px traveled, the walk frame advances (toggling between frame 0 and 1). Direction flips at viewport edges — or instantly when the mouse passes over the critter.

### PRNG — mulberry32

```js
function mulberry32(seed) {
  return () => {
    seed = (seed + 0x6d2b79f5) | 0;
    let t = Math.imul(seed ^ seed >>> 15, 1 | seed);
    t = (t + Math.imul(t ^ t >>> 7, 61 | t)) ^ t;
    return ((t ^ t >>> 14) >>> 0) / 4294967296;
  };
}
```
