# the_office
Chrome extensions with everyday functionality inspired by actors from The Office.

## Install

Each project is a standalone Chrome extension. To load any of them:

1. Open `chrome://extensions` in Chrome
2. Enable **Developer mode** (toggle in the top-right)
3. Click **Load unpacked** and select the project folder

---

## Projects

### pixel-tab-art
Injects a unique pixel-art sprite into every tab so you can identify which tab you're on at a glance. The art is deterministic — seeded from the page's hostname — so the same site always gets the same sprite. Sprites use a mirrored 8×8 grid with a procedurally generated palette, and the overlay is draggable and toggleable via the extension popup.

See [`pixel-tab-art/README.md`](./pixel-tab-art/README.md) for full details.
